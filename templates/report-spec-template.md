# Report Specification Template

Use this template to define report requirements before or during development.

## Report Name
Example: Executive Operations Dashboard

## Audience
Example: Executive leadership, program managers, operations analysts

## Business Objective
Example: Provide a clear view of operational performance, backlog, and KPI trends.

## Core KPIs
- Completion Rate
- Open Cases
- Average Resolution Days
- Backlog Volume

## Required Visuals
- KPI cards
- trend chart
- regional breakdown
- detail table

## Filters
- Date
- Region
- Status
- Priority

## Refresh Cadence
Example: Daily

## Validation Notes
- Confirm KPI totals match source data
- Validate filter behavior
- Review labels with business stakeholders
