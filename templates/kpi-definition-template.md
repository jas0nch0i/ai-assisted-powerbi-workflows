# KPI Definition Template

Use this template to define report KPIs in a consistent, business-friendly way.

| KPI Name | Formula | Business Meaning | Owner | Caveats | Validation Method |
|---|---|---|---|---|---|
| Example: Completion Rate | Completed Items / Total Items | Shows percent of work completed | Operations Lead | Zero denominator should return 0 | Compare to filtered detail totals |
| Example: Open Cases | Count of cases with status = Open | Shows active work volume | Program Manager | Excludes cancelled records | Validate against source status counts |

## Instructions

- Use plain English for business meaning.
- Document any assumptions or caveats.
- Include a simple validation step whenever possible.
