# BI Prompt Patterns

This file contains reusable prompt examples for common BI development tasks.

## DAX Prompts

- Review this DAX measure and explain why the totals may be incorrect.
- Rewrite this measure for readability and explain the logic.
- Suggest a better way to calculate rolling 12-month performance.

## KPI Definition Prompts

- Draft a business-friendly KPI definition for executive reporting.
- Rewrite this KPI so a non-technical stakeholder can understand it.
- Create a KPI definition table using name, formula, meaning, owner, and caveats.

## Documentation Prompts

- Generate a data dictionary for the following fields.
- Draft semantic model documentation for these tables and relationships.
- Write a report specification for an operations dashboard.

## Dashboard Design Prompts

- Suggest a dashboard layout for an executive audience.
- Recommend KPI grouping for a performance reporting page.
- Simplify this dashboard concept into a cleaner one-page structure.

## Troubleshooting Prompts

- Identify likely causes of incorrect totals in this matrix visual.
- Suggest validation steps for a DAX measure that changes unexpectedly with filters.
- Create a test checklist for KPI validation before release.
